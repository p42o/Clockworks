Questions

Answered.

What does "bring your own API key" mean?

You sign up at console.anthropic.com, add a payment method, and copy your API key into Sniper's Settings. You pay Anthropic directly for each generation — typically $0.10–$0.50 per site. We don't touch your billing.

Does this run in the cloud?

No. Clockworks Sniper runs entirely on your Mac — it's a local web app that opens in your browser. The only external calls are to the Anthropic API (for generation) and optionally Replicate (for AI photos). Your target site data never hits our servers.

What do I actually deliver to the client?

You get a ZIP with: (1) a PIN-locked preview HTML file you send to the prospect, (2) a clean handoff HTML file you deliver after payment, and (3) an AI-generated onboarding form so they can supply their real content. Hosting is the client's problem — you're selling the build.

How much should I charge clients?

Most local small business sites (landscaping, pest control, cleaning, trades) sell for $1,000–$3,000 for a one-pager. Multi-page sites run $3,000–$7,000. Sniper generates a single-page site — position it as a foundation and upsell from there. One sale pays for the tool 10–30×.

Do I need to know how to code?

No coding required to generate and export sites. You need Python 3 on your Mac (it's either already there or a 2-minute install). The output is clean HTML/CSS you or a developer can customize further.

What if the generated site isn't perfect?

The Prompt Review step lets you add custom instructions before generating — "add a gallery section," "use a dark theme," "make the hero focus on emergency availability." You can regenerate as many times as you want. Each attempt costs ~$0.25.

What happens after I buy?

You'll receive a download link for the Clockworks Sniper ZIP. Unzip it, double-click Launch Sniper.command to start, and open your browser to localhost:5555. First run takes about 30 seconds to set up dependencies.

Is this legal? Can I scrape any site?

You're responsible for how you use the tool. Scraping publicly visible information is generally legal, but using it to deceive, spam, or misrepresent a business is not.
